# Quiz Console App
A Java-based console application that allows users to attempt multiple-choice quiz questions. 
This project is built to practice core Java and object-oriented programming concepts.
## Features
- Console-based quiz application
- Multiple-choice questions
- Object-oriented design
- Beginner-friendly code
## Technologies Used
- Java
- Core OOP Concepts
- Console I/O
## Project Structure
Quiz Console App
├── Main.java
├── Question.java
└── QuestionService.java
## How to Run
1. Clone the repository
2. Open terminal in project folder
3. Compile:
   javac *.java
4. Run:
   java Main
## Learning Outcomes
- Java classes and objects
- Arrays of objects
- Method overriding
- Console-based applications
